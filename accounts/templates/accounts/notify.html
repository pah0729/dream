Pushbullet 다운로드 및 설정방법

* 푸시토큰 변경시에도 같은 방법으로 설정하세요.


  • 1. Pushbullet 사이트 접속 및 로그인

    • https://www.pushbullet.com/

    • Pushbullet 사이트 접속 후 오른쪽 상단에 있는 'Sign in' 또는 'sign up with Google' 버튼을 눌러 로그인합니다.

  • 2. Settings에 있는 'create Access Token' 클릭

    • 로그인 후 화면 왼쪽 메뉴 맨 아래에 있는 Settings 클릭 후 'Create Access Token' 버튼을 누릅니다.

  • 3. Access Token 복사

    • 버튼을 누른 후 바로 위에 뜬 검정 상자에 적힌 토큰을 복사합니다.

  • 4. 복사한 Access Token 설정

    • 마이페이지 우측 '설정 및 관리' 화면의 '푸시토큰 추가' 버튼을 클릭하여 복사한 토큰을 븥여넣고 저장합니다.

  • 5. Chrome 웹 스토어 접속

    • 다시 왼쪽 메뉴에서 Devices를 누른 후 Chrome을 선택하여 'Chrome 웹 스토어' 에 접속합니다.

  • 6. Chrome에 추가

    • 'Chrome에 추가' 버튼을 눌러 Pushbullet을 추가합니다.

  • 6. 모바일 애플리케이션 추가

    • 웹 뿐만 아니라 모바일에서도 알람을 받을 수 있습니다.
      안드로이드 사용자는 구글플레이, 아이폰 사용자는 앱스토어에서 'Pushbullet'을 다운받아 사용하시면 됩니다.
